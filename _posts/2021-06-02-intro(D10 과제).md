[ 문제상황 1 ]

가위, 바위, 보 게임을 만들어 봅시다. 가위바위
보 게임은 다음과 같이 진행됩니다. ex1)
나 입력(가위:s, 바위:r, 보:p): s
너 입력(가위:s, 바위:r, 보:p): s
비겼다. ex2)
나 입력(가위:s, 바위:r, 보:p): r
너 입력(가위:s, 바위:r, 보:p): s
이겼다. ex3)
나 입력(가위:s, 바위:r, 보:p): p
너 입력(가위:s, 바위:r, 보:p): s
졌다.

[ 소스코드 1 ]
~~~python
import random
state_list = ['s','p','r']


while True:
  state = input('입력(가위:s, 바위:r, 보:p): ')
  counter_state = random.choice(state_list)

  print('[ 나 : {} ]'.format(state))
  print('[ 컴퓨터 : {} ]'.format(counter_state))

  if state == counter_state:
    print('비겼다.')

  elif (state == 's' and counter_state == 's') or (state == 'p' and counter_state == 'r') or (state == 'r' and counter_state == 's'):
    print('이겼다.')
  else:
    print('졌다.')
~~~

[ 결과 1 ]
~~~python
입력(가위:s, 바위:r, 보:p): p
[ 나 : p ]
[ 컴퓨터 : p ]
비겼다.
~~~





[ 문제상황 2 ]

놀이 공원의 자유이용권 가격
을 계산하는 프로그램을 작성하려
고 합니다. 요금표는 아래와 같
고, 고객의 나이와 입장 시간에
따라 금액이 달라집니다. 출력 예
를 참고하여 구입요금을 계산하는
프로그램을 작성하시오. 단, 시간
은 24시간제를 적용하므로 오후
5시는 17시로 표현한다.

[ 소스코드 2 ]
~~~python
time = int(input('시간 : '))
age = int(input('나이: '))

if (24>time>=17):
  print('10,000')
elif (0<=time<17) and (age>=65 or 3<=age<=12):
  print('25,000')
elif (0<=time<17) and (13<age<64):
  print('34,000')
~~~
