[ 문제상황 ]

```python
NAVER에서는 주기적으로 비밀번호를 변경하여 개인정보를 안전하게 보호하고 있습니다. 
ID는 ‘알파벳+숫자’의 조합으로 만들었다고 할 때, 
새로 변경하려는 비밀번호가 ID의 숫자부분의 패턴과 3글자 이상 일치한다면 그것은 사용이 허락되지않습니다.
위에서 설명한 것과 같이 비밀번호 변경 가능 여부를 체크하는 프로그램을 작성하시오.
```



[ 소스코드 ]
```python
#Q1

ID = input('[ 아이디 ] :')
PASSWORD = input('[ 비밀번호 ] :')

state = False

Empty_list_A = []
Empty_list_B = []

length_PASSWORD = len(PASSWORD)
length_ID = len(ID)

Available_cnt = 15 #비밀번호 한계자리점 : 15 자리

ID_listed = list(ID)
PASSWORD_listed = list(PASSWORD)
for cnt in range(3, Available_cnt):

  for a in range(0, length_PASSWORD):
    for n in range(0, length_PASSWORD):
      if len(str(PASSWORD[a:a+n])) == cnt:
        Empty_list_A.append(PASSWORD[a:a+n])

  for a in range(0, length_ID):
    for n in range(0, length_ID):
      if len(str(ID[a:a+n])) == cnt:
        Empty_list_B.append(ID[a:a+n])   

for n in Empty_list_B:
  for a in Empty_list_A:
    if a == n:
      state = True

if state != True:
  print("비밀번호를 변경할수 있습니다.")
else:
  print("비밀번호를 변경할수 없습니다.")
```
